# My Typora Themes

这个仓库存放我日常使用的两套 Typora 主题配置：

- **drake-custom.css**：来源于 [liangjingkanji/DrakeTyporaTheme](https://github.com/liangjingkanji/DrakeTyporaTheme)，我在原版基础上去掉了背景网格，保留其提示块、表格和 JetBrains Mono/iconfont 资源，适合偏冷色调、无衬线的阅读体验。
- **zj-custom.css**：来源于 [Theigrams/My-Typora-Themes](https://github.com/Theigrams/My-Typora-Themes)，保留原作者的红色系、衬线字体以及编辑/导出模式的差异化体验，同时继续使用 `zj/` 目录下的字体资源。
